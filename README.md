# Dino Runner 3D

Dino Runner 3D is a small Unity 3D endless runner inspired by the classic browser dinosaur game.

The player controls a dinosaur running through a stylized desert environment. The goal is to jump over cactus obstacles, survive as long as possible, and earn points while the game speed gradually increases.

## Features

- 3D endless runner gameplay
- Dinosaur jump mechanic
- Cactus obstacle spawning
- Score and high score system
- Increasing game speed over time
- Start menu and Game Over screen
- Stylized low-poly desert environment

## Controls

- Space — Jump
- Enter — Start game
- R — Restart after Game Over
- Esc — Exit or pause, if available

## How to Open

1. Install Unity Hub.
2. Add this project folder through Unity Hub.
3. Open the main scene from `Assets/Scenes`.
4. Press Play in the Unity Editor.
